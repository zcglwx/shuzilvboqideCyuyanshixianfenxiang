## 内容概述

- **低通滤波器**：实现了基本的低通滤波器，适用于需要去除高频噪声的应用场景。
- **带通滤波器**：实现了带通滤波器，适用于需要保留特定频率范围内的信号的应用场景。
- **高通滤波器**：实现了高通滤波器，适用于需要去除低频噪声的应用场景。
- **巴特沃斯滤波器**：实现了巴特沃斯滤波器，提供了平滑的频率响应，适用于对频率响应要求较高的应用场景。
